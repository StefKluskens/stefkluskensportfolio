Mega Man

A Mega Man clone that I made for a school programming assignment.
This was the first game I made coding in C++. In a couple of months I learned how to use the given framework and create a replica of the Bomb Man stage in Mega Man.
Things I learned during this project:
- How to implement a camera from scratch
- How to animate sprites using code
- How to collision checks using SVG's
