Gladiator Fighter


Gladiator Fighter was the game I made for my first bachelorâ€™s degree as my grad work.

This game was developed using the Unity game engine and was made to run on the Oculus Quest 2. Because this project was meant for my grad work, I wanted to do something new and challenging.

I had just purchased an Oculus Quest 2 headset and wanted to make a game for it, so this was the perfect opportunity to do so.
This was my first attempt at a virtual reality game and I learned a lot from it.

Things I learned during this project:
- How to port a game to a device other than a computer
- How to link Unity and the Oculus Quest 2 headset
- How to optimize a game for it to run standalone on the headset
